The data generator program is a Java application designed to supply the system with realistic, sample data for testing and showcase purposes.

## Requirements
* Installed **JVM 8+**
* A **MSSQL Server 2012** +

## Features
* Create a new client
* View clients
* View orders
* Create an invoice
* Export the invoice to a .pdf file
